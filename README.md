# 📱 Mobile Sales Dashboard – Power BI

## Overview
This project presents an interactive **Power BI Mobile Sales Dashboard** that analyzes mobile phone sales performance across different brands, cities, payment methods, and customer demographics. The dashboard helps businesses understand sales trends, customer purchasing behavior, and brand performance through dynamic visualizations and KPIs.

---

## 🎯 Objectives

- Monitor overall mobile sales performance.
- Analyze sales by different mobile brands and models.
- Track Month-to-Date (MTD) sales performance.
- Compare current sales with the Same Period Last Year (SPLY).
- Understand customer purchasing behavior across cities.
- Analyze payment method preferences.
- Evaluate customer ratings and sales trends.
- Support business decisions through interactive insights.

Provides a complete overview of:

- Total Sales

- Total Transactions
- Average Sales
- Brand Performance

- Payment Method Distribution

---

### 2️⃣ MTD Report

Displays Month-to-Date business performance including:

- Current Month Sales
- MTD Quantity
- MTD Transactions
- Daily Sales Trend
- Brand Performance (Current Month)

---

### 3️⃣ Same Period Last Year (SPLY)

Compares current performance against the same period from the previous year.

Includes:

- Sales Comparison
- Quantity Comparison
- Transaction Comparison
- Growth Analysis
- Performance Variance

## 📌 How to Use

1. Download the repository.
2. Open the **.pbix** file using **Power BI Desktop**.
3. Refresh the dataset if necessary.
4. Use the slicers to interact with the dashboard.
5. Hover over charts to view detailed insights.
6. Export reports for presentations or business analysis.
